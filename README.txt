




How to pull latest version:

	run the Git Bash.vbs is PortableGit_93... folder
	cd to autolock

	git pull





Instructions for commit to git:

	run the Git Bash.vbs is PortableGit_93... folder
	cd to autolock

	git pull
	git add . 
	git commit -m "<whatever you want for a message in quotes>"
	git push



Undo local changes (go back to last commited version)
	make sure you're in the repo, then run:

	git checkout -- .
